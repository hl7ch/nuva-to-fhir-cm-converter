# NUVA CVS to FHIR ConceptMap Converter

## Swiss vaccine codes
The converter is converting the CSV files from NUVA to FHIR ConceptMap files.
You can find the NUVA source [here](https://github.com/IVC-NUVA/NUVA/)

Implemented is:
- NUVA to Swissmedic
- Swissmedic to NUVA
- NUVA to Swisslegacy
- Swisslegacy to NUVA
